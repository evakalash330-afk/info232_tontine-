# 💰 TontiPredict

##  Description
TontiPredict est une application web développée avec Streamlit permettant de collecter, analyser et prédire les comportements des membres d'une tontine (épargne communautaire).

Ce projet s'inscrit dans le cadre du cours INF232 - Analyse de données (EC2).

# Objectifs
- Collecter les données des membres
- Analyser les comportements financiers
- Prédire les retards
- Segmenter les membres

## Fonctionnalités
- Formulaire de collecte
- Visualisation des données
- Régression linéaire
- Classification (Random Forest)
- Clustering (KMeans)

## Technologies
- Python, Streamlit, Pandas, NumPy, Scikit-learn, Plotly

## Lancement
pip install -r requirements.txt
streamlit run app.py
